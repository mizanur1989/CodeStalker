# Команды CodeStalker

## Основные команды

### Сканирование и разведка
- **`nmap [опции] [цель]`** — Сканирование сети и поиск уязвимых хостов
  - `nmap -sV 192.168.1.0/24` — Версионное сканирование подсети
  - `nmap -p 80,443 target.com` — Сканирование конкретных портов
  - `nmap -A 192.168.1.45` — Агрессивное сканирование (OS, версии, скрипты)

### Взлом и эксплуатация
- **`hydra -l логин -P пароли.txt цель`** — Brute-force атака
  - `hydra -l admin -P passwords.txt 192.168.1.45 ssh` — Взлом SSH
  - `hydra -l admin -P passwords.txt target.com http-post-form` — Взлом веб-формы

- **`sqlmap -u "URL" --dbs`** — SQL-инъекции
  - `sqlmap -u "http://target.com/login.php" --dbs` — Перечисление баз данных
  - `sqlmap -u "URL" -D database --tables` — Перечисление таблиц
### Криптография
- **`openssl enc -d -aes-256-cbc -in файл.enc -k пароль`** — Расшифровка AES
- **`john hash.txt`** — Взлом хешей паролей (John the Ripper)
- **`hashcat -m 0 -a 0 hashes.txt wordlist.txt`** — Взлом MD5 хешей

### Информационные команды
- **`whoami`** — Показать информацию о профиле игрока
- **`help`** — Показать справку по игре
- **`about`** или **`profile`** — Информация об авторе
- **`exit`** или **`quit`** — Выход из терминала

### Продвинутые команды (для высокого ранга)
- **`metasploit`** — Фреймворк для эксплуатации уязвимостей
- **`wireshark`** — Анализ сетевого трафика
- **`burpsuite`** — Прокси для веб-атак
- **`aircrack-ng`** — Взлом Wi-Fi сетей

## Примеры использования

### Миссия 1: Сетевая разведка
$ nmap -sV 192.168.1.0/24
[Сканер] [██████████████████████████] 100%
Обнаружены узлы:
├── 192.168.1.45    - Веб-сервер (Apache 2.4.41) [Уязвим]
└── 192.168.1.123   - База данных (MySQL 5.7.30) [Слабый пароль]
### Миссия 2: Взлом паролей
$ hydra -l admin -P passwords.txt 192.168.1.45 ssh
[STATUS] 145.234 tasks remaining
[ssh] host: 192.168.1.45   login: admin   password: P@ssw0rd123

## Система рангов

| Ранг | Прогресс | Доступные инструменты |
|------|----------|----------------------|
| **Новичок** | 0-25% | `nmap`, `whoami`, `help` |
| **Хактивист** | 25-50% | `hydra`, `sqlmap` |
| **Пентестер** | 50-75% | `metasploit`, `wireshark` |
| **Кибер-бог** | 75-100% | Все инструменты + zero-day |

---

## Советы

1. **Начинай с разведки** — всегда используй `nmap` для поиска целей
2. **Собирай информацию** — читай вывод команд, ищи подсказки
3. **Не торопись** — у каждой миссии есть таймер
4. **Проверяй прогресс** — команда `whoami` покажет твой статус
5. **Экспериментируй** — пробуй разные варианты команд и параметров
